# dungeon-machine

Dungeon Machine is meant to provide a multiplayer tabletop RPG experience having AI perform the role of the game master.
This involves granting players a complete freedom in acting within the world of the game as well as allowing them to request a campaign in a chosen setting (e.g. high fantasy, cyberpunk).

### To run the game
```bash
cd ./game
npm run dev
```

### To run backend
```bash
cd ./backend
uv run fastapi dev app/main.py
```
